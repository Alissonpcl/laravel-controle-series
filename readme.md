### Subindo para producao

1. Atualizar o **APP_KEY** com o comando abaixo `php artisan key:generate`

1. Ajustar os seguintes parametros no*.env*:
    1. APP_ENV=prod
    1. APP_DEBUG=false
1. Ajustar demais parametros no **.env** conforme necessidade
